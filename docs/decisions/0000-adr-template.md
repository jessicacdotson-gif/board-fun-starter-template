# 0000 — ADR template

**Status:** template (copy me — or run `/new-decision`)
**Date:** YYYY-MM-DD

## Context

What situation forced a decision? Include the constraint that made it
non-obvious. Link the inbox entry or spec question it came from.

## Decision

One sentence. What we're doing.

## Reasoning

Why this over the alternatives — name the alternatives and why each
lost. Future-you will re-ask this exact question; answer it now.

## Consequences

What this commits us to, what it costs, what becomes easier/harder.
Include the reversal condition if there is one ("revisit if X").

---

Rules: one file per decision, numbered in order. Superseded ADRs get a
"Superseded by NNNN" line added — never rewrite their content; ADRs
are history, not documentation of the present (that's `docs/spec/`).
